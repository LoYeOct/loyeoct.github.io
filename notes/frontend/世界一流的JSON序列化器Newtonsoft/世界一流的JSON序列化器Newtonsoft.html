        # 世界一流的JSON序列化器Newtonsoft

        # 写文原由

        相信大家都对json有自己的认识，规范也是早就定义好的了，无需多谈，但是尽管知道，例如团队中有几个人都知道服务端需要返回json的序列给到客户端接收，客户端看不出问题，需要的参数整整齐齐一个不落，但是一看后端程序不禁感叹，勤恳是真的勤恳，放个图感受一下。

        没错！就是字符串拼接，这个一行放一个参数拼接已经算不错的了，某种意义上代码还是工整的，因为肯定少不了一行两三个拼接参。这样拼接只有一个用处就是让代码能跑起来，这样拼接着实有够累人的（看着都累），容错性还有排错难易度就更不用谈了。那么diss就到这，开始介绍今天的主角“Newtonsoft”

        # Newtonsoft用法简介

        谷歌搜一下“Newtonsoft”，官网首页就很明显的放上了用法图，还有一些性能介绍，本文标题也是直接通过机翻介绍取名的，就很实(tou)在(lan),"高效的性能"，“兼容性”，“最受欢迎的.net库”等等，足以证明“Newtonsoft”的强大，不仅强大，使用也特简单方便，下面引用了官网的三种使用示例还有自己的实调案例。

        1、Serialize JSON（序列化JSON）

        Product product = new Product();
product.Name = "Apple";
product.Expiry = new DateTime(2008, 12, 28);
product.Sizes = new string[] { "Small" };

string json = JsonConvert.SerializeObject(product);
// {
//   "Name": "Apple",
//   "Expiry": "2008-12-28T00:00:00",
//   "Sizes": [
//     "Small"
//   ]
// }

        2、Deserialize JSON（反序列化JSON）

        string json = @"{
  'Name': 'Bad Boys',
  'ReleaseDate': '1995-4-7T00:00:00',
  'Genres': [
    'Action',
    'Comedy'
  ]
}";

Movie m = JsonConvert.DeserializeObject<Movie>(json);

string name = m.Name;
// Bad Boys

        3、LINQ to JSON（LINQ转JSON）

        JArray array = new JArray();
array.Add("Manual text");
array.Add(new DateTime(2000, 5, 23));

JObject o = new JObject();
o["MyArray"] = array;

string json = o.ToString();
// {
//   "MyArray": [
//     "Manual text",
//     "2000-05-23T00:00:00"
//   ]
// }

        4、easy demo

         private string test()
        {//测试
            var abcObject = new
            {
                uid = "11",
                name = "test",
                keyword = "newtonsoft",
                insertDate = DateTime.Now
            };
            string serJson = JsonConvert.SerializeObject(abcObject);
            return "[" + serJson+","+serJson +"]";//返回单个json序列直接返回serJson，多个的话就稍微拼接一下，用for循环serJson在[]里面，记得初始化
        }

        # 结语

        使用简单，好用还高效，何乐而不为。很多时候偷懒才是人进步源泉，抱着一颗“嗯，还有更简单的方法”去钻研，夜深了，下一篇再见，晚安。
